## Install live-server
```
npm install -g live-server
```

## Run live server:
```
live-server
```


## Running MyApp
cd into MyApp directory
Run:
```
npm install
```

Still in MyApp directory run:
```
npm run dev
```

## Visit site in browser:
visit:
```
http://localhost:3000
```

## Running json server to simulate a production rest server locally:
npm run json-server
